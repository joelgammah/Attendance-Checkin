Introduction

Welcome to the Attendance Check-In user documentation. These pages explain how to use the web app to create events, generate QR codes, and check in attendees. The docs are bundled into the frontend and served at /docs.

What you'll find here

  • How to log in and access the dashboard
  • How to create events
  • How to check in attendees

Editing these docs

Documentation source files live under frontend/public/docs/. For production updates edit these files, then rebuild and push the frontend Docker image (see Build & Push guidance in RENDER.md), and trigger a redeploy on Render.
